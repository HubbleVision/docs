# Transaction / Balance

本页作为交易与余额 API 的概览页，汇总端点与请求/响应形态说明。示例全部放在各自的 endpoint 页面。

## 基础信息
- 基础路径：`https://api.hubble-rpc.xyz/`
- 认证方式：`HUBBLE-API-KEY: <your-key>`（Header，区分大小写）

<a id="request-response-variants"></a>
## 请求/响应形态说明
- 认证：所有端点均需在请求头携带 `HUBBLE-API-KEY: <your-key>`。
- 请求体差异：
  - 交易列表：包含分页字段 `page`、`pageSize`，以及查询条件（例如 `symbol` 等）。
  - 余额查询：指定 `walletAddress`（地址）与 `tokenAddress`（可选，代币 mint）。
- 统一响应与错误：
  - （空）

## 端点概览
- `/tx/api/v1/sol/tx`：按条件分页获取交易列表（POST）
- `/balance/api/v1/sol/balance`：获取指定地址/代币余额（POST）

> 详细参数与示例请进入左侧导航中的对应 endpoint 页面。
