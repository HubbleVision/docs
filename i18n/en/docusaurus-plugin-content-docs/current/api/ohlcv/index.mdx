# OHLCV (K-line Data)

This page is an overview for the OHLCV API. It consolidates endpoints and request/response shapes. Examples are provided on each endpoint page.

## Basics
- Base URL: `https://api.hubble-rpc.xyz/candle/api/v1`
- Authentication: `HUBBLE-API-KEY: <your-key>` (Header, case-sensitive)
