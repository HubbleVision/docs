# Transaction / Balance

This page is an overview for the Transaction & Balance APIs. It consolidates endpoints and request/response shapes. Examples live on their endpoint pages.

## Basics
- Base URL: `https://api.hubble-rpc.xyz/`
- Authentication: `HUBBLE-API-KEY: <your-key>` (Header, case-sensitive)

<a id="request-response-variants"></a>
## Request/Response shapes
- Authentication: every endpoint requires the `HUBBLE-API-KEY: <your-key>` header.
- Request body differences:
  - Transactions list: includes pagination fields `page`, `pageSize`, and query filters (e.g., `symbol`).
  - Balance query: provide `walletAddress` (address) and `tokenAddress` (optional, token mint).

## Endpoints
- `/tx/api/v1/sol/tx`: get transactions with filters and pagination (POST)
- `/balance/api/v1/sol/balance`: get balance for address/token (POST)
